# 常用命令

## nest cli 命令

```bash
# 增加模块
$ nest g mo xxx
$ nest g module xxx
$ nest g mo server/xxx

# 增加控制器
$ nest g co xxx
$ nest g co server/xxx

# 增加服务
$ nest g service xxx
$ nest g service server/xxx
```
